Automatically stores uploads in uniqid folders, renames files, repairs old media, updates database URLs, and regenerates thumbnails.

== Description ==

- Rename uploaded files to uniqid
- Store uploads in uniqid-based folders
- Migrate old uploads
- Fix broken media URLs
- Regenerate thumbnails
- Fully repair database links

== Installation ==

1. Upload plugin folder
2. Activate plugin
3. Go to UniqID Media menu
4. Click Repair & Migrate
